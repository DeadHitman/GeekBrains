3 урок курс 2

Integer,Char,Float,String,Boolean,Long,Byte - это immutable объекты - неизменяемые
Дженерики - это обобщенное программирование

Iterator -превращается в foreach
foreach - создает кучу раз String в стеке
Stack -первым зашел, последним вышел (LIFO - последним зашел, первым вышел)
в ArrayList - имеем быстрый доступ,но медленное удаление (если элементы удаляются не из конца списка)
LinkedList - доступ медленно, а удаление и добавление быстро
hashcode - адрес объекта
по equals - объекты будут равны,если их значения совпадают

HashSet - во множествах нет повторяющихся элементов (hash  -не должен у них совпадать)
сортировка в HashSet - идет по hashcode
Hashset - нет индекса, нет повторений, не гарантировано последовательность добавления
TreeSet - сортирующее бинарное дерево поиска (сортируется так как нам надо) в алфавитном порядке
нет повторений,гарантировано сортируется
как сравнить коробки? Comparable -  интерфейс надо имплеменить в коробки
и переопределить метод compareTo

HashMap - в отличие от Set ,значения ключей перезаписываются при дублировании ключа
не гарантируется порядок при вывода на экран
TreeMap - сортирует ключи (по алфавиту если стринг)
